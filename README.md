#  Simulador de Folha de Pagamento 2026

##  Sobre o projeto

Este projeto foi desenvolvido como atividade prática do 4º semestre do curso de Gestão da Tecnologia da Informação, na disciplina de Programação para Internet, ministrada pelo professor Ricardo Leme.

O objetivo do trabalho foi aplicar conceitos fundamentais de desenvolvimento web utilizando HTML5, CSS3 e JavaScript, por meio da criação de um simulador de folha de pagamento. O sistema permite ao usuário inserir o valor do salário bruto e, a partir disso, realiza automaticamente o cálculo dos descontos de INSS de forma progressiva e do IRRF com base na legislação vigente de 2026, além de apresentar o salário líquido final.

##  Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

##  Funcionalidades
- Cálculo automático de INSS (progressivo)
- Cálculo de IRRF
- Exibição do salário líquido
- Mensagem de isenção de imposto
